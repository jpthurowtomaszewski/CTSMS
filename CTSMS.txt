Comandos Úteis GIT:

git init = inicia o git na pasta

git status = exibe as informações dos arquivos da pasta

git add "nomedoarquivo.exe" = adiciona o arquivo aos arquivos a serem enviados

git add . = adiciona todos arquivos em vermelho no "git status" para serem adicionados nos arquivos a serem enviados.

git commit -m "mensagem aqui" = prepara um commit para o github com todos arquivos adicionados

git push = envia para o github os arquivos preparados no commit

git reflog = mostra as versões disponíveis deste programa

git reset --hard 53x54c2 = (o código é um exemplo de código de versão), isso reseta a aplicação para a versão selecionada

git branch = mostra as branchs disponiveis

git branch nomedabranch = cria uma branch
